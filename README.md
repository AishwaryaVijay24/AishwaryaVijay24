## 🚀 Selected Work

### 🔬 SciClaim — Scientific Claim Verification

Built a retrieval and generation pipeline for scientific claim verification using:

* BM25 retrieval
* Dense retrieval
* Hybrid search
* Query rewriting
* Reranking
* LLM-based structured outputs
* Evaluation using Recall and nDCG

This project helped me understand how retrieval quality, evidence grounding, and evaluation metrics directly affect the reliability of AI-generated answers.

---

### 🧩 L&D Nexus — Full Stack & AI Development

Worked on AI-driven classification and matching systems for a live platform, focusing on:

* Converting product requirements into ML/system workflows
* Building backend pipelines for experimentation and deployment
* Improving outputs through evaluation and error analysis
* Working with noisy and incomplete real-world data
* Communicating technical decisions to technical and non-technical stakeholders

This strengthened my ability to build practical AI features that work beyond prototypes.

---

### 📄 CyberResponse — IEEE-Published Research

Contributed to **CyberResponse: A Resilient Framework for Collecting and Responding to Digital Crimes and Frauds**, published on **IEEE Xplore**.

My work involved AI-related system design, feature engineering, evaluation, and research communication for real-world cybercrime and digital fraud response.

---

### 🌱 EcoEats — Sustainability-Focused Application

Designed and developed a sustainability-focused application aimed at reducing food waste, adapted for the Indian context.

This project helped me understand how technology adoption depends not only on technical design, but also on social context, data availability, and user needs.
